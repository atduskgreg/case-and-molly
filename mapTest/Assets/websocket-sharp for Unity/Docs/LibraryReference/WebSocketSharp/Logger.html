Logger Class

Provides the simple logging functions.

Syntax

public class Logger

Remarks

The Logger class provides some methods that output the logs associated with the each WebSocketSharp.LogLevel values. If the WebSocketSharp.LogLevel value associated with a log is less than the Logger.Level, the log can not be outputted.

The default output action used by the output methods outputs the log data to the standard output stream and writes the same log data to the Logger.File if it has a valid path.

If you want to run custom output action, you can replace the current output action with your output action by using the Logger.SetOutput(Action<LogData, string>) method.
